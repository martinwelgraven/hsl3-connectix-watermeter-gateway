1. Description

This Logic Node connects to the Connectix Watermeter Gateway and retrieves various data such as watermeter values, firmware information, and leak detection status.

2. Inputs

No. Name Initialisation Description
1 Gateway Host Address connectix_watermeter.local
The host address of the Connectix Watermeter Gateway.
2 Gateway Port 82
The port of the Connectix Watermeter Gateway.
3 Read Interval 60
The interval at which the Connectix Watermeter Gateway reads data.

3. Outputs

No. Name Initialisation SBC Description
1 MAC Address 00_00_00_00_00_00 s
The MAC address.
2 Gateway Model connectix_watermeter_gateway_v1.0 - A s
The name of the model.
3 Startup Time 5-01-2026 06:03:16 s
A timestamp of when the gateway started up.
4 Firmware Running 03-07-2025 v1 s
The current firmware version.
5 Firmware Available (Y/N) 03-07-2025 v1 s
The latest firmware available.
6 Firmware Update Available (0 = False, 1 = True) 0 s
Indication on availability of firmware update (0 = False, 1 = True).
7 WiFi RSSI 0 s
The WiFi signal strength (RSSI).
8 MQTT Configured (0 = False, 1 = True) 0 s
Indication of whether MQTT is configured (0 = False, 1 = True).
9 Watermeter Pulse Factor 0 s
The pulse factor.
10 Watermeter Value 0 s
The latest current watermeter value read.
11 Watermeter Used Last Minute 0.0 s
The amount of water used in the last minute.
12 Watermeter Pulse Count 0 s
The total pulse count of the watermeter.
13 Leak Detected (0 = False, 1 = True) 0 s
Indication of whether a leak is detected (0 = False, 1 = True).
s = send, sbc = send by change
